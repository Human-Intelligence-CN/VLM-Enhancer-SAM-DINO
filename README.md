# VLM-Enhancer--SAM-DINO-
本项目旨在增强现有视觉语言大模型（VLM），结合 GroundingDINO 与 SAM2 技术，实现图像与文本的精确分析与理解。通过 GroundingDINO，模型能够根据文本指令识别图像中的任意对象，生成精确的检测框；SAM2 提供可提示式实例分割及视频记忆机制，使目标在视频中保持时空一致性分割。结合 InstructBLIP，项目支持指令微调与视觉语义提取，提升模型在零样本场景下的表现。整体方案实现了从图像编码、跨模态特征增强、目标定位到大语言模型生成回答的完整流程，为复杂视觉任务提供精准、高效的解决方案，适用于图像问答、视频分析及多模态信息处理等应用场景。
